## 프로젝트 소개

- 저희 팀은 남녀노소 가릴 것 없이 모든 사람들이 쉽게 접할 수 있는 것이 무엇이 있을까? 라고 고민한 결과 부담없이 매일 어디서든 할 수 있는 운동 러닝을 생각했습니다. 목표 설정부터 활동기록까지 이제 막 러닝을 시작했거나 개인 기록갱신을 목표로 하는 사람들을 도와주고자 러닝캘린더를 만들게 되었습니다.
